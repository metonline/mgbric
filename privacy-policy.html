Gizlilik Politikası

Hoşgörü Turnuva Analiz Uygulaması ("Uygulama"), gizliliğinize saygı duymaktadır. Bu Gizlilik Politikası, uygulamayı kullanırken verilerinizin nasıl toplandığını, kullanıldığını ve korunduğunu açıklamaktadır.

1. Toplanan Veriler

1.1 Otomatik Olarak Toplanan Veriler

  • Cihaz Bilgileri: İşletim sistemi, cihaz modeli ve tarayıcı türü
  • Kullanım Verileri: Uygulamada hangi özellikleri kullandığınız, sayfalar arasında nasıl gezindiğiniz
  • Konum Verileri: Yerel saatiniz (turnuva sonuçlarını yerel saate göre göstermek için)

1.2 Siz Tarafından Sağlanan Veriler

  • Arama Terimi: Turnuva veya oyuncu ararsanız arama teriminiz
  • Tercih Seçimleri: Dil tercihleriniz (Türkçe veya İngilizce)

1.3 Otomatik Olarak Toplanmayan Veriler

Biz ad, e-posta, telefon numarası veya finansal bilgi toplamıyoruz. Uygulama tamamen anonim bir şekilde çalışır.

2. Verilerin Kullanımı

Toplanan veriler aşağıdaki amaçlarla kullanılır:

  • Turnuva sonuçlarını göstermek ve analiz etmek
  • Uygulamayı geliştirebilmek ve hataları düzeltmek
  • Kullanıcı deneyimini iyileştirmek
  • Uygulamanın performansını izlemek
  • Yasal yükümlülükleri yerine getirmek

3. Veri Depolanması

3.1 Yerel Depolama

Dil tercihiniz ve bazı kullanıcı ayarları tarayıcınızın yerel depolamasında saklanır. Bu veriler yalnızca sizin cihazınızda bulunur.

3.2 Sunucu Depolaması

Turnuva sonuçları GitHub Pages'te barındırılan JSON veritabanından getirilir. Kişisel bilgiler depolanmaz.

3.3 Veri Kaynağı

Turnuva verileri Vugraph.com'dan otomatik olarak güncellenir. Vugraph'ın Gizlilik Politikasını buradan inceleyebilirsiniz.

4. Üçüncü Taraf Hizmetleri

Uygulama aşağıdaki üçüncü taraf hizmetleriyle entegre olmuştur:

  • GitHub Pages: Web sitesi barındırması için
  • Vugraph API: Turnuva verileri almak için

Bu hizmetlerin kendi Gizlilik Politikaları vardır. Detaylar için lütfen onların sayfalarını ziyaret edin.

5. Veri Güvenliği

Verilerinizi korumak için aşağıdaki önlemleri almıştır:

  • HTTPS kullanan güvenli bağlantı
  • Service Worker tarafından veriler çevrimdışı mod için kaşelenir
  • Kişisel bilgiler toplanmaz
  • Veriler üçüncü taraflarla paylaşılmaz

6. Çerezler (Cookies)

Uygulama, çerezler kullanmadığı için rıza talebinde bulunmaz. Tüm veriler tarayıcı yerel depolamasında saklanır.

7. Çocukların Gizliliği

Uygulama, 13 yaşından küçük çocukları hedeflemez ve bilerek 13 yaşından küçük çocuklardan veri toplamaz.

8. Haklarınız

Aşağıdaki haklara sahipsiniz:

  • Cihazınızdan tarayıcı depolamasını silmeyi seçebilirsiniz
  • Dil tercihini istediğiniz zaman değiştirebilirsiniz
  • Uygulamayı istediğiniz zaman silebilirsiniz
  • Topladığımız veriler hakkında bilgi talep edebilirsiniz

9. Bu Politikada Değişiklikler

Bu Gizlilik Politikasını değiştirme hakkını saklı tutarız. Değişiklikler bu sayfada yayınlandığında geçerli olur. Önemli değişiklikler için e-posta bildirimi sağlanacaktır.

Privacy Policy

The Hoşgörü Tournament Analysis Application ("Application") respects your privacy. This Privacy Policy explains how your data is collected, used, and protected when you use the application.

1. Data Collected

1.1 Automatically Collected Data

  • Device Information: Operating system, device model, and browser type
  • Usage Data: Which features you use, how you navigate between pages
  • Location Data: Your local time (to display tournament results according to your local time)

1.2 Data You Provide

  • Search Terms: If you search for tournaments or players
  • Preference Choices: Your language preferences (Turkish or English)

1.3 Data We Do NOT Collect

We do NOT collect names, email addresses, phone numbers, or financial information. The application operates completely anonymously.

2. Use of Data

Collected data is used for the following purposes:

  • Display and analyze tournament results
  • Improve and fix errors in the application
  • Enhance user experience
  • Monitor application performance
  • Fulfill legal obligations

3. Data Storage

3.1 Local Storage

Your language preference and some user settings are stored in your browser's local storage. This data only exists on your device.

3.2 Server Storage

Tournament results are retrieved from a JSON database hosted on GitHub Pages. No personal information is stored.

3.3 Data Source

Tournament data is automatically updated from Vugraph.com. You can review Vugraph's Privacy Policy here.

4. Third-Party Services

The application is integrated with the following third-party services:

  • GitHub Pages: For web hosting
  • Vugraph API: To retrieve tournament data

These services have their own Privacy Policies. Please visit their pages for more information.

5. Data Security

We take the following measures to protect your information:

  • Secure connections using HTTPS
  • Data is cached by Service Worker for offline mode
  • No personal information is collected
  • Data is not shared with third parties

6. Cookies

The application does not use cookies and therefore does not require consent. All data is stored in browser local storage.

7. Children's Privacy

The application is not intended for children under 13 and does not knowingly collect information from children under 13.

8. Your Rights

You have the following rights:

  • You can choose to delete browser storage from your device
  • You can change your language preference at any time
  • You can delete the application at any time
  • You can request information about the data we collect

9. Changes to This Policy

We reserve the right to modify this Privacy Policy. Changes become effective when published on this page. Email notifications will be provided for significant changes.
